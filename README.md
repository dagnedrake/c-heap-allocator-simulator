# C Heap Allocator Simulator

A small C heap allocator simulator using boundary tags, block splitting, and coalescing.

## Status

This repository is currently unfinished.

The core heap simulator code is present, along with a simple demo driver in main() that 
allocates and frees blocks while printing the memory pool after each operation. 
The project still needs cleanup, boundary checks, organized tests, and improved documentation.
